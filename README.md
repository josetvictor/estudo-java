# Repositório de estudo com Java

Este repositório é dedicado ao estudo e prática de Java, abrangendo desde conceitos básicos até tópicos avançados. Ele inclui exemplos de código, exercícios e projetos para aprimorar minhas habilidades em Java.

## Motivação

A motivação para criar este repositório é organizar meu aprendizado de Java de forma estruturada e compartilhar minha evolução. Além disso, quero ter esse repositório como referência futura para todos meus aprendizados com essa tecnologia.

## Estrutura do Repositório

O repositório está organizado da seguinte forma:
- `CRS`: *contém arquivos e projetos relacionados à realização de cursos, com foco no aprimoramento profissional e técnico.*
- `Recursos`: *reúne testes de pacotes, bibliotecas, arquivos e ferramentas úteis para a framework ou linguagem utilizada.*
- `Padrões`: *aborda a experimentação e aplicação de diferentes padrões de projeto, visando aprimorar a qualidade e a estrutura do código.*
- `Tecnologia`: *projetos para exercício de diferentes tecnologias que a framework oferece, como APIs, webclients e etc.*
- `Projetos`: *projetos mais complexos que integram um ou mais recursos das pastas anteriores.*
